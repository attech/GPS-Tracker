Herzlich Willkommen zu Android GPS Tracking App

Berechtigungen

android.perrmission.Fine_Location
    Wird benötigt um den genauen GPS Standort zu ermitteln.
android.perrmission.Corsare_Location
    Wird benötigt um den ungefährem Standort per Internet/W-LAN zu ermitteln.
android.permission.INTERNET
    Wird benötigt um ihre Daten auf den Server zu Laden.

Beschreibung der App

Mit dieser App kannst du ganz einfach deinen Sport verwalten, einfach App starten und auf Neu klicken, damit wird deine Route mithilfe von GPS aufgezeichnet. Sobald du das Aufzeichnen beendest, werden dir ein paar fragen gestellt (Temperatur, Wetter, Gefühl), danach wird alles auf den Server hochgeladen (nur wenn eine Internetverbindung zur Verfügung steht).

Ich wünsche euch viel Spaß mit dieser App.
